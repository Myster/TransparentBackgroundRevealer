TransparentBackgroundRevealer
=============================

Bookmarklet to reveal transparent backgrounds by applying a checkerbaord behind.

To make this bookmarklet work, create a bookmark with this url
```
javascript:(function(){document.body.appendChild(document.createElement('script')).src='https://rawgit.com/Myster/TransparentBackgroundRevealer/master/transparent-background-revealer.js';})();
```
